Actualizar datos

Los datos se gestionan directamente en Google Sheets. Para actualizar, edita el Sheet de la sede correspondiente y luego haz clic en Sincronizar en la barra superior.

Flujo de actualización
1Descarga el reporte del LMS en formato CSV
2Abre el Google Sheet y reemplaza los datos en la hoja correspondiente
3Regresa aquí y haz clic en Sincronizar — todos los dispositivos se actualizan
Credenciales Google
Google Client ID
Google Cloud Console → Credenciales → OAuth 2.0 Client ID → Aplicación web
Google Sheet ID
El ID en la URL del Sheet: docs.google.com/spreadsheets/d/ESTE-ID/edit
Nombre de la hoja
La pestaña del Sheet con los datos (por defecto: Hoja1)
Nombre hoja — Lima Linings
La pestaña del Sheet con los datos de Lima Linings
Pasos en Google Cloud Console
01console.cloud.google.com → Nuevo proyecto → nombre: bradken-auth
02APIs y servicios → Biblioteca → habilitar Google Sheets API y Google Drive API
03APIs y servicios → Pantalla de consentimiento OAuth → Externo → completar nombre y correo
04Credenciales → + Crear credencial → ID de cliente OAuth 2.0 → Aplicación web
05Orígenes JS autorizados: https://TU-USUARIO.github.io y http://localhost
06Copiar Client ID → pegarlo en "Google Client ID" arriba → Guardar
07Crear Google Sheet con columnas: id nombre cargo curso estado valido_hasta → copiar ID de la URL
08Compartir el Sheet: "Cualquiera con el enlace puede ver" (para que la sincronización pública funcione)
